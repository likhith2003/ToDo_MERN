## A todo list made using MERN stack.

1. Clone / Download the repository
```bash
git clone git@github.com:likhith2003/ToDo_MERN.git
```

2. Install dependencies

   ```bash
   npm install
   cd client
   npm install
   ```

3. Build the frontend

   ```bash
   yarn build
   ```

4. Run the server

   ```bash
   cd ..
   npm start
   ```
